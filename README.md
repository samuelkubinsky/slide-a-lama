# Slide-a-Lama

- console / web UI
- written in Java, HTML and CSS
- game for 2 players
- scores, comments and ratings are storing to PostgreSQL
- implements - JUnit, Maven, Spring Boot, JDBC, JPA, Rest, Thymeleaf
